# Markup 擴展指南

本指南說明如何使用 USD Explorer 模板中的 Markup 擴展來添加註解和標記，用於審查會話。

## 概述

Markup 擴展提供在審查會話期間添加註解的功能。它允許審查者在場景中添加視覺標記和文字註解，這些標記可以保存、導航和導出。

## 步驟 1: 準備場景

### 開啟 Factory.usd

1. **開啟應用程式**
   - 啟動您的 USD Explorer 應用程式

2. **載入場景**
   - 如果尚未開啟，開啟 `Factory.usd` 檔案
   - 可以在 Examples 標籤中找到此檔案

### 進入 Review 模式

1. **切換到 Review 模式**
   - 點擊應用程式頂部的 **Review** 標籤
   - 這會切換到 Review 模式

2. **了解 Review 模式**
   - Review 模式包含底部中央的**導航欄（Navigation Bar）**
   - 導航欄可以通過右下角的切換按鈕開啟或關閉

## 步驟 2: 開啟 Markup 面板

1. **導航到測量位置**
   - 使用視窗導航工具移動到場景中的測量位置
   - 或使用之前創建的測量點

2. **開啟 Markup 面板**
   - 點擊導航欄上的 **Markup** 按鈕
   - Markup 面板會開啟

## 步驟 3: 啟用 Markup 功能

1. **啟用 Markup**
   - 在 Markup 面板中點擊 **Add Markup** 按鈕
   - 這會啟用標記功能

2. **準備繪製**
   - 游標會改變，表示可以開始繪製標記

## 步驟 4: 繪製標記

1. **繪製圓圈**
   - 在測量位置周圍**拖動游標**繪製一個圓圈
   - 圓圈會標記您想要註解的区域

2. **調整標記**
   - 可以調整圓圈的大小和位置
   - 確保標記清楚標示目標區域

## 步驟 5: 添加文字註解

1. **開啟註解功能**
   - 點擊 **Comment** 按鈕

2. **選擇顏色**
   - 點擊**紅色顏色選擇器**
   - 將顏色更改為**亮藍色**（或其他您喜歡的顏色）
   - 顏色選擇器通常在工具欄或面板中

3. **創建文字框**
   - 在測量位置附近**點擊**創建文字框
   - 文字框會出現在點擊位置

4. **輸入註解**
   - 在文字框中**輸入您的註解**
   - 例如："需要檢查此測量值"

5. **應用註解**
   - 點擊文字框**外部**以應用註解
   - 註解會保存並顯示在場景中

## 步驟 6: 保存 Markup

1. **應用 Markup**
   - 在 Markup 工具欄上點擊 **Apply** 按鈕
   - 這會保存當前的標記和註解

2. **查看保存的 Markup**
   - 標記現在會保存在 **Markup 列表**中
   - 可以在 Markup 視窗中查看所有保存的標記

## 步驟 7: 導航到保存的 Markup

### 方法 1: 從 Markup 視窗

1. **開啟 Markup 視窗**
   - 如果未開啟，點擊 Markup 按鈕

2. **選擇 Markup**
   - 在 Markup 列表中點擊保存的標記
   - 視窗會自動導航到該標記位置

### 方法 2: 從視窗雙擊

1. **在視窗中找到標記**
   - 標記會顯示為視覺元素（圓圈和註解）

2. **雙擊標記**
   - 在視窗中**雙擊**標記
   - 視窗會聚焦到該標記

## 導出 Markup

### 導出功能

Markup 可以導出到外部文件格式：

1. **PDF 格式**
   - 導出為 PDF 文件
   - 適合分享和列印

2. **CSV 格式**
   - 導出為 CSV 文件
   - 適合數據分析和報告

### 導出步驟

1. **開啟導出選項**
   - 在 Markup 視窗中找到導出選項
   - 通常在 File 或 Export 選單中

2. **選擇格式**
   - 選擇 PDF 或 CSV 格式

3. **選擇保存位置**
   - 選擇文件保存位置
   - 點擊保存

## Markup 功能特性

### 視覺標記

- **圓圈標記**：標記特定區域
- **箭頭**：指向特定位置
- **高亮**：突出顯示重要區域

### 文字註解

- **顏色自訂**：選擇不同的顏色
- **位置調整**：可以移動註解位置
- **多行文字**：支援多行註解

### 組織功能

- **標記列表**：所有標記保存在列表中
- **快速導航**：點擊標記快速跳轉
- **分類標記**：可以為標記添加類別或標籤

## 工作流程提示

### 1. 審查會話

- 在 Review 模式中使用 Markup
- 為每個問題或觀察添加標記
- 使用不同顏色區分不同類型的註解

### 2. 協作審查

- 導出 Markup 為 PDF 分享
- 使用 CSV 格式進行數據分析
- 保存場景以保留所有標記

### 3. 組織標記

- 使用有意義的註解文字
- 為標記添加時間戳記（如果支援）
- 定期導出和備份標記

## 常見問題

### Q: Markup 按鈕沒有出現？

**A**: 
- 確認已進入 Review 模式
- 檢查導航欄是否已開啟
- 確認 Markup 擴展已啟用

### Q: 無法保存 Markup？

**A**: 
- 確保點擊了 Apply 按鈕
- 檢查場景是否已保存
- 確認有寫入權限

### Q: 如何刪除 Markup？

**A**: 
- 在 Markup 列表中選擇標記
- 使用刪除按鈕或 Delete 鍵
- 或右鍵點擊選擇刪除選項
