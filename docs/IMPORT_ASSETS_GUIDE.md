# 導入資產指南

本指南說明如何在您的 USD Explorer 應用程式中從各種來源導入 3D 資產，包括 CAD 繪圖、製造商模型或資產庫。

## 概述

終端使用者會從各種來源導入 3D 資料：
- CAD 繪圖
- 製造商模型
- 資產庫（如 NVIDIA Assets）

本指南將示範如何從 NVIDIA Assets 庫導入資產。

## 步驟 1: 啟動應用程式

### 從 VS Code 啟動

1. **在 VS Code 中開啟專案**
   - 開啟包含您的 USD Explorer 應用程式的專案資料夾

2. **啟動應用程式**
   - 使用 VS Code 的終端或命令面板啟動應用程式
   - 或直接從 Omniverse Launcher 啟動

### 從 Omniverse Launcher 啟動

1. 開啟 **Omniverse Launcher**
2. 找到您的自訂應用程式
3. 點擊 **Launch** 啟動應用程式

## 步驟 2: 載入 Factory_Lite.usd

1. **開啟檔案**
   - 在應用程式啟動後，選擇 **File > Open**
   - 或點擊歡迎畫面中的 **Open** 按鈕

2. **選擇檔案**
   - 瀏覽到 `Factory_Lite.usd` 檔案位置
   - 選擇檔案並點擊 **Open**

3. **確認載入**
   - 檔案載入後，您應該能在視窗中看到工廠場景

## 步驟 3: 存取 NVIDIA Assets 庫

1. **找到 NVIDIA Assets 標籤**
   - 在應用程式視窗的**左下角**找到 **NVIDIA Assets** 標籤
   - 如果標籤未顯示，可能需要展開左側面板

2. **點擊 Warehouse 選項**
   - 在 NVIDIA Assets 面板中，點擊 **Warehouse** 選項
   - 這會顯示倉庫相關的資產庫

## 步驟 4: 瀏覽和選擇資產

1. **滾動瀏覽選項**
   - 在 Warehouse 資產庫中滾動瀏覽可用的資產
   - 資產可能包括：
     - 貨架和儲存設備
     - 搬運設備
     - 包裝材料
     - 其他倉庫相關物件

2. **選擇要添加的資產**
   - 找到您想要添加到場景中的資產
   - 考慮資產的用途和位置

## 步驟 5: 拖放資產到場景

1. **拖動資產**
   - 從資產瀏覽器中**拖動**選定的資產
   - 將資產拖向**視窗（Viewport）**

2. **查看放置指示器**
   - 當資產被拖入視窗時，會顯示**十字準線（cross hairs）**
   - 十字準線指示資產將放置在**地板**上的位置
   - 移動滑鼠可以調整放置位置

3. **釋放資產**
   - 當位置合適時，**釋放滑鼠按鈕**
   - 資產會自動放置在**地平面**上
   - 資產會自動對齊到地面

## 資產放置特性

### 自動對齊功能

- **地平面對齊**：資產會自動放置在場景的地平面上
- **位置指示**：十字準線提供即時的位置反饋
- **即時預覽**：拖動時可以看到資產的預覽位置

### 放置後調整

資產放置後，您可以：
- **移動**：使用移動工具調整位置
- **旋轉**：使用旋轉工具調整方向
- **縮放**：使用縮放工具調整大小
- **刪除**：如果不需要，可以刪除資產

## 工作流程提示

### 1. 規劃資產放置

在添加資產之前：
- 考慮場景的整體布局
- 規劃資產的位置和方向
- 確保資產不會與現有物件重疊

### 2. 使用多個資產

- 可以從資產庫添加多個資產
- 每個資產都會自動對齊到地平面
- 使用移動工具調整相對位置

### 3. 組織場景

- 使用 USD 的圖層功能組織資產
- 為不同類型的資產創建不同的圖層
- 使用命名約定保持場景整潔

## 下一步：複製資產

在下一步中，您將學習如何使用 OpenUSD 的方法來**複製資產**。

### 複製資產的方法

OpenUSD 提供多種複製資產的方法：

1. **直接複製**
   - 在場景中選擇資產
   - 使用複製/貼上功能

2. **實例化（Instancing）**
   - 創建資產的實例
   - 節省記憶體和效能

3. **引用（References）**
   - 引用原始資產
   - 保持與原始資產的連結

## 常見問題

### Q: 找不到 NVIDIA Assets 標籤？

**A**: 
- 檢查左側面板是否已展開
- 確認應用程式已正確載入所有擴展
- 嘗試重新啟動應用程式

### Q: 資產無法正確放置？

**A**:
- 確保場景中有地平面
- 檢查資產的座標系統是否正確
- 使用移動工具手動調整位置

### Q: 如何刪除已添加的資產？

**A**:
- 選擇資產
- 按 `Delete` 鍵或使用編輯選單中的刪除選項

### Q: 可以導入自訂資產嗎？

**A**: 可以！
- 使用 **File > Import** 導入自訂資產
- 支援多種格式（USD, FBX, OBJ 等）
- 導入的資產會自動轉換為 USD 格式
