# Measure 擴展指南

本指南說明如何使用 USD Explorer 模板中的 Measure 擴展來測量場景中的距離。

## 概述

Measure 擴展是 USD Explorer 模板中包含的範例擴展之一，提供測量場景中距離、角度等功能。

## 步驟 1: 啟動應用程式並載入場景

1. **啟動應用程式**
   - 從 VS Code 或 Omniverse Launcher 啟動您的應用程式

2. **開啟新場景**
   - 點擊 **New** 開啟 Layout 模式
   - 或使用 File > New

3. **載入 Factory.usd**
   - 在應用程式底部找到 **Examples** 標籤
   - 找到 **Factory.usd** 檔案
   - 開啟該場景

## 步驟 2: 找到要測量的資產

### 使用搜尋功能

1. **開啟 Stage 視窗**
   - 如果未顯示，從 Window 選單開啟 Stage 視窗

2. **搜尋資產**
   - 在 Stage 視窗的搜尋框中輸入 **"robotpedestal"**
   - 這會過濾顯示名稱中包含該詞的資產

3. **選擇資產**
   - 在搜尋結果中選擇 **RobotPedestal**
   - 如果路徑太長，可以：
     - 調整 Stage 視窗的欄寬
     - 或檢查 Property 面板確認選擇的資產

### 聚焦資產

1. **聚焦視圖**
   - 選擇 RobotPedestal 後，按 **F 鍵**
   - 這會將視窗聚焦到該資產

2. **導航視圖**
   - 使用視窗導航工具調整視角
   - 確保可以清楚看到機器人基座及其旁邊的基座

## 步驟 3: 開啟 Measure 工具

1. **開啟 Measure 視窗**
   - 從應用程式選單選擇 **Tools > Measure**
   - Measure 視窗會開啟

2. **了解 Measure 功能**
   - Measure 擴展可以測量：
     - 表面之間的距離
     - 頂點之間的距離
     - 邊緣之間的距離
     - 角度等

## 步驟 4: 執行測量

### 使用頂點測量

1. **選擇測量模式**
   - 在 Measure 視窗的 **Placement** 區段
   - 選擇 **Vertex** 選項

2. **開始測量**
   - 點擊第一個機器人基座的**角落頂點**
   - 這會開始測量

3. **完成測量**
   - 將游標移動到另一個基座的頂點
   - 點擊該頂點
   - Measure 工具會顯示兩個基座之間的距離

### 測量結果

- 距離會以**數值**和**視覺線條**顯示
- 可以在 Measure 視窗中查看詳細資訊
- 測量結果可以保存供後續參考

## 步驟 5: 保存場景

1. **保存場景**
   - 選擇 **File > Save As**
   - 使用新的檔名保存場景
   - 這將用於下一步的標記（Markup）功能

## Measure 擴展的功能

### 測量類型

1. **頂點測量（Vertex）**
   - 測量兩個頂點之間的距離
   - 適合精確測量特定點

2. **表面測量（Surface）**
   - 測量兩個表面之間的距離
   - 自動選擇最近的表面點

3. **邊緣測量（Edge）**
   - 測量兩個邊緣之間的距離
   - 適合測量結構元素

### 進階功能

- **角度測量**：測量角度
- **面積測量**：測量表面積
- **體積測量**：測量體積（如果支援）

## 工作流程提示

### 1. 精確測量

- 使用頂點模式進行最精確的測量
- 確保視圖清晰，可以準確點擊目標點
- 使用聚焦功能（F 鍵）確保看到正確的資產

### 2. 多個測量

- 可以進行多個測量
- 每個測量會顯示不同的顏色或標籤
- 在 Measure 視窗中管理所有測量

### 3. 保存測量結果

- 測量結果會保存在場景中
- 保存場景以保留測量資料
- 可以導出測量報告

## 下一步：使用 Markup 擴展

在下一步中，您將學習：
- 使用 Markup 擴展標記測量結果
- 為審查添加註解和標記

## 常見問題

### Q: Measure 視窗沒有出現？

**A**: 
- 確認 Measure 擴展已啟用
- 檢查 Window 選單中是否有 Measure 選項
- 嘗試重新啟動應用程式

### Q: 無法準確點擊頂點？

**A**: 
- 放大視圖以更清楚地看到頂點
- 使用頂點捕捉功能（如果可用）
- 嘗試使用表面模式代替

### Q: 測量結果不正確？

**A**: 
- 確認單位設定正確
- 檢查場景的座標系統
- 確保點擊的是正確的點
